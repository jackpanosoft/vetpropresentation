Our Journey

From a DOS application to a national cloud platform, the evolution of VetPro has been driven by innovation and a commitment to serving veterans.

1991 - VetProDOS

  • VetProDOS is developed by Panoramic Software for California County Veteran Service Offices to manage claims and file VA claims on the behalf of Veterans

2000 - VetProSQL

  • VetPro system is transitioned from a DOS application to Microsoft Windows and SQL server platform.

2005 - VetProWeb is LIVE!

  • VetProWeb is launched as a cloud based web application.
  • 15 counties converted with 49 users.

2010-2013 - VetProWeb for CALVET

  • All California Counties migrated to use a single application for claims and reporting.
  • Onboarding results in 57 agencies/400+ users.
  • A single application allows CalVet to ensure quality and consistency in claims processing.
  • VetProWeb also allows CalVet to track county submission data to inform distribution of subvention funds to counties based on claims filed and awarded.

2013-2016 - Continued VetProWeb Enhancements

  • System evolves from filing claims via fax to electronic submission.
  • District Office review process is implemented to help ensure accuracy and completeness of claims.
  • Additional Features incrementally added to the VetProWeb application

2015 - The Plan for the Future

  • Panoramic assesses need to move all applications to a new platform for continued growth, innovation, and stability.
  • VetProWeb is selected as the pilot product to be modernized.
  • Due to annual funding limitations, work is split into what can be completed each year with the available funding.

2016-2017 - Foundations of the New Platform

  • Requirements Gathering and Analysis
  • Cloud Infrastructure
  • Architecture of Application Environment
  • Technology Stack Selection (Elm, PostGreSQL, Slate, Haskell)
  • Database and Backup Processes
  • Base Application (Menus, Grids, Reporting Tools)

2017-2018 - Core Functionality

  • Prototype Design & Review
  • VA Form Analysis and Rendering Functionality
  • File Upload Functionality
  • User Management
  • Office Management
  • Case Management
  • Benefit Management

2018-2019 - Launch Preparation

  • Direct Submission
  • Infrastructure
  • Business Logic
  • Begin Form Development

April 2019 - VetPro National Launched

  • VetPro National launched on April 1, 2019, with LaSalle County, Illinois as the first to go live and still use VetPro today.
  • This national platform allows us to generate revenue while refining the application for the eventual CalVet migration.

2019-2020 - Pandemic Response & Growth

  • Accreditation Management and Direct Submit Management
  • Signature portal
  • Built at the onset of the COVID-19 pandemic to allow VSOs to continue serving Veterans remotely.
  • First custom reports module for South Carolina
  • American Legion for the State of CA came onboard
  • March 2020: VetPro Made Available for Free During Covid.

2020-2021 - Additional CalVet State Specific Features

  • College Fee Waiver Verification Portal
  • Agency Attachments
  • County Form Letter Management
  • Medi-Cal Avoidance Tracking
  • Base State Application

2021-2023 - Key Partnerships

  • Dale K. Graham Foundation Onboarding
  • The first custom forms module was launched for NM.

2023-2024 - Renewed Momentum

  • Form Letters
  • Reception Queue

2025 - AutoEstablish Release

  • The release of AutoEstablish marks a significant milestone in automating and streamlining the benefits application process for veterans.

Simplified View of VetPro Configurations

A high level view of the VetPro configurations currently in use and the plan for California's migration.

Standard VetPro

  • Most customers have a single instance of VetPro in which users manage veteran information, complete forms, obtain signatures via the signature portal, and submit forms to the VA's Lighthouse API.

TVC VetPro

  • TVC VetPro is a VetPro configuration used by the Texas Veterans Commission.
  • It is a single instance of VetPro used by multiple agencies across the state of Texas
  • This is our largest single VetPro instance.

South Carolina VetPro

  • South Carolina VetPro is a VetPro configuration used by the South Carolina Department of Veterans Affairs.
  • Each county in South Carolina has it's own VetPro instance.
  • A central application pulls information from each of these instances to populate a customized suite of reports.

New Mexico VetPro

  • New Mexico VetPro is a VetPro configuration used by the New Mexico Department of Veterans Affairs.
  • It is a single instance of VetPro used by multiple counties across the state of New Mexico.
  • A separate application houses a series of custom forms specific to New Mexico.

Plan for California VetPro

Visualizing the migration from the current legacy system to the future VetPro National environment.

CalVet Legacy VetProWeb

  • Legacy VetProWeb is the legacy system used by the California Department of Veterans Affairs.
  • It is a single instance of VetPro used by multiple counties across California.
  • It was developed as an offshoot from our Public Guardian/Public Administration applications and features were added incrementally over the years.
  • As with VetPro National, the core of the application involves managing veteran information, completing forms, obtaining signatures via the signature portal, and submitting forms to the VA's Lighthouse API.
  • The application supports a review process where regional offices can review and approve forms before submission to the VA.
  • It allows the state to manage county subventions and track county performance via an auditing process.
  • It also interfaces with a College Fee Waiver Verification Portal for secure management of College Fee Waivers.

CalVet VetPro National

  • CalVet VetPro National is the fulfilment of the original VetPro plan started in 2015.
  • Each county will have their own instance of VetPro National.
  • A central CalVet application will be used to manage county subventions, track county performance, and manage compliance documentation.
  • Some part of the CalVet ecosystem will also need to interface with the College Fee Waiver Verification Portal currently used by Legacy VetProwWeb.

California VetPro Migration

  • Fulfillment of the original VetPro plan started in 2015
  • Each county will have their own instance of VetPro National
  • Central CalVet application for county subvention management and compliance tracking
  • Integration with College Fee Waiver Verification Portal
  • Modern, scalable architecture supporting future growth
